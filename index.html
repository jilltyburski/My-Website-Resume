Notes on Design Process

I decided to write all of the HTML and CSS for this website so that I could proudly say that I created my resume website myself. I want my resume website to still have the clarity and readability of a traditional resume but with added touches of design and responsiveness that a website can offer.

I chose to use Bootstrap’s framework because its grid system is an efficient way to make a website responsive. It is likely that most viewers of this website would be on a desktop or larger screen, but I still wanted to make sure that it is easy to view on mobile or tablet displays as well.

I created the geometric background design for the header in Adobe Illustrator. I am inspired by geometric patterns I have seen on other websites and wanted to try to make my own version of it.

My goal for the work experience section is to highlight the main skills I gained from various jobs I have had. I utilized jQuery to allow for the viewer to hover and click on the broader skill for each job to reveal more specific examples.
